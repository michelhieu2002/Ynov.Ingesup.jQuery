AJAX & jQuery - Les Evénements

  1. Aller sur cette page: events.html
  2. Ouvrir le fichier js/events.js en édition

  3. Ajouter les fonctionnalités suivantes:

    1. Survol d'une ligneAjoute/Supprime la classe CSS 'highlight' à la ligne survolée
    2. Filtre sur la DateAffiche les lignes correspondantes lors du changement de DateAttribut 'data-date'
    3. Filtre sur le SportAffiche les lignes correspondantes lors du changement de SportAttribut 'data-sport'
    4. Filtre sur le MatchAffiche les lignes correspondantes lorsque l'utilisateur saisit le match
    5. Affichage des InformationsAffiche les informations du match lors du clic sur une ligne
