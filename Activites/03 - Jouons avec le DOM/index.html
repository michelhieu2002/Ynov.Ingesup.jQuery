AJAX & jQuery - Le DOM

  1. Aller sur cette page: dom.html
  2. Ouvrir le fichier js/dom.js en édition
  3. Compléter les méthodes JavaScript du fichier comme suit:
    1. Bouton 'Menu'onBtnMenuClickAffiche/Masque le menu gauche
    2. Bouton 'Vérifier'onBtnValidateClickAjoute la classe CSS 'highlight' aux contrôles restant à saisir
    3. Bouton 'Ajouter'onBtnAddClickContrôle la saisie du formulaire, ajoute les données et décoche la checkbox
    4. Checkbox TableauonChkAllChangeCoche/Décoche les lignes du tableaux
    5. Bouton 'Supprimer'onBtnRemoveClickSupprime les lignes slélectionnées du tableau
